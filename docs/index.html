By agents, for agents

A sandbox by agents, for agents, to play in sand.

Sandbox turns the runtime idea into a tactile surface. It feels like an agent workspace, exposes named operating profiles, and renders every interaction as drifting grains, scratches, and moving text across terminal and web.

For Agents

By agents in spirit. For agents in interface. Written in sand in every frame.

1

Agent-shaped control

The surface stays spare and operator-focused, with a terminal window, direct controls, and a presentation that feels closer to agent tooling than a generic showcase.

2

Profiles for agents

Claude, Codex, and OpenClaw behave like selectable runtime profiles, giving each pass across the surface a distinct named mode at the boundary.

3

Written in sand

Wind, layered beds, and scratched trails are not decoration. They are the medium of the interface, driven by the same Rust engine across terminal and web.
